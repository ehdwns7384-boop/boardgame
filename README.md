# 블클타 도우미

오프라인 모임에서 같은 와이파이를 쓰는 플레이어들이 휴대폰으로 접속하는 Blood on the Clocktower 보조 앱입니다.

## 실행

```powershell
py app.py
```

서버가 켜지면 화면에 `스토리텔러 PIN`과 접속 주소가 표시됩니다.

- 스토리텔러: 노트북에서 접속 후 PIN 입력
- 플레이어: 같은 와이파이에서 접속 주소로 들어가 닉네임 입력

## 첫 버전 기능

- 5~15명 플레이어 참가 제한
- Trouble Brewing 역할 자동 배정
- 주정뱅이의 실제 역할/보이는 역할 분리
- 스토리텔러 그리모어
- 스토리텔러 원형 보드와 토큰 클릭 1:1 메시지
- 생존, 유령표, 중독, 취함, 보호 상태 관리
- 낮/밤 전환
- 실시간 지명/투표/처형 후보 관리
- 지명자 좌석부터 시작하는 3초 준비 + 15초 시계방향 자동 투표
- 플레이어 능력 요청과 스토리텔러 비밀 메시지 전달

## 메모

이 버전은 게임 규칙을 완전 자동 판정하지 않고, 스토리텔러의 판단을 빠르게 기록하고 전달하는 보조 도구로 설계되어 있습니다.
